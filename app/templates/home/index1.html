Welcome to the label site

In this webpage, you are going to find the applicable rules to describe how this complex sentence becomes a simple sentence. The first table shows a pair of sentences, the first line is the complex sentence, and the second line is the simplified sentence. The second table gives information about changed words(exist in the simple sentence but not in the complex sentence) and six words in the complex sentence related to each changed word. Then, there are two tasks for you to do: a world-level change one and a grammar-level change one.
For the word-level task, you are going to select a pair of words(The first word is one of the six relevant rank words and the second word is the changed word). This pair of words can indicate how the sentence is simplified. For example, one complex word is replaced by a simple word) or one is the pronoun of the other. In addition, there are some automated generate choices provided. if these choices are matched with the change in the sentence, you can select them. You also can add some additional word pairs in the input box. (the input format is the same as the automated generate choice)
For the grammar level task, you are going to select a pair of phrases(The first phrase contains one of the six relevant rank words, and [part of speech tagger] is the same as the syntactic structure in the complex sentence, and another phrase contains the changed words, and [part of speech tageers] is the same as the structure in the simplified sentence ). The pair of phrases can indicate how the sentence is simplified. Compared with the first task, this task you are going to focus more on the syntactic part. There are some automated generate choices provided. if these choices are matched with the change in the sentence, you can select them. You also can add some additional phrases pair in the input box(the input format is the same as the automated generate choice)
